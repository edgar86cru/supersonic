# Reactive&nbsp;Data

`steroids.data.reactive` provides cues for when to update your views with fresh data.


## Properties

- `steroids.data.reactive.whenVisible`


# steroids.data.reactive.whenVisible

A stream of notifications for when the current `WebView` becomes visible.

## Methods

- `whenVisible.onValue(callback)`

Provide a callback function to listen to an event in the stream.
