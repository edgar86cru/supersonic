# steroids.modal

A representation of a modal window that can be shown on top of the whole application.
Only one modal window can be active at a time.

Modal window shows any `steroids.views.*` object.

## Supported platforms

- iOS
- Android

## Methods

- steroids.modal.show
- steroids.modal.hide
- steroids.modal.hideAll
- steroids.modal.on
- steroids.modal.off
