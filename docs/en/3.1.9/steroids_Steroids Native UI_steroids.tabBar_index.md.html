steroids.tabBar

The native tab bar needs to be enabled by defining steroids.config.tabBar in config/application.coffee. By default, the tab bar that is shown on the screen in all views, with each tab pointing to a separate WebView. See the config/application.coffee guide for more information. Note that the tab bar is also available on Android, but cannot be shown, hidden or updated programmatically.

Methods

  • steroids.tabBar.hide
  • steroids.tabBar.show
  • steroids.tabBar.update
